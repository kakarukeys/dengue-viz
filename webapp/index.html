What caused Singapore dengue outbreak?

This is a tool for analyzing the cause of Singapore dengue outbreak with the data collected from NEA website and other geospatial datasets available on the internet. We hope that with the help of this tool, response effort can be more targetted and future dengue outbreak can be better managed.

This beautiful visualization is the result of the volunteer work of a few software developers.

If you want to see Singapore dengue-free in future, you may donate some money to help keep this project going.
